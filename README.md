# Pygame Arcanoid 🎮🔨

Динамичный проект классической аркады, созданный на мощном игровом движке **Pygame**. В проекте реализована механика уничтожения объектов, физика движения мяча и система управления платформой.

## 🚀 Особенности реализации
- **Классовая структура:** Проект построен на наследовании классов (`Area` -> `Picture`, `Area` -> `Label`), что делает код расширяемым.
- **Игровая физика:** Реализованы алгоритмы отскока мяча от стен, потолка и подвижной платформы.
- **Механика разрушений:** Коллекция "монстров" хранится в динамическом списке, объекты удаляются при столкновении с мячом.
- **Условия победы/поражения:** Автоматическое определение конца игры при потере мяча или полной очистке поля.
- **Управление:** Плавное движение платформы с помощью обработки событий клавиатуры.

## ⌨️ Управление
- **Стрелка Вправо (Right):** Движение платформы вправо.
- **Стрелка Влево (Left):** Движение платформы влево.
- **Цель:** Уничтожить все иконки монстров на поле.

## 🛠 Технологии
- **Python 3.x**
- **Pygame** (библиотека для разработки игр)

## 🔧 Запуск и ресурсы
Для запуска требуются изображения: `platform.webp`, `ball.webp`, `enemy.webp`.
```bash
pip install pygame
python Arcanoid.py
```
